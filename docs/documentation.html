class s3m.Connection(path, lock_transactions=True, lock_timeout=-1, *args, **kwargs)¶

Implements functionality of both a connection and a cursor. It won’t let multiple database operations execute in parallel. It can also block parallel transactions (with lock_transactions=True).

Parameters:
  • path – Path to the database
  • lock_transactions – If True, parallel transactions will be blocked
  • lock_timeout – Maximum amount of time the connection is allowed to wait for a lock. If the timeout is exceeded, LockTimeoutError will be thrown. -1 disables the timeout.
